La Théorie des Nombres Typographiques de Douglas Hofstadter

Référence : Douglas Hofstadter, Gödel, Escher, Bach, les Brins d'une Guirlande Eternelle, InterEditions, Paris.

Calcul propositionnel

  • Expressions bien formées
    • Atomes : P, Q, R, P', Q', R', P'' ...
    • x, y -> ~x, <x & y>, <x V y>, <x => y>

  • Règles
    • Adjonction : x, y |- <x & y>
    • Séparation :
      • <x & y> |- x
      • <x & y> |- y
    • Double tilde : ~~ peut être supprimée ou introduite si le résultat est bien formé.
    • Fantaisie : [x ... y ...] |- <x => y>
    • Report : x |- [y ... x ] (Tout théorème de la réalité de niveau supérieur peut être introduit dans une fantaisie)
    • Détachement : x, <x => y> |- y
    • Contraposition : <x => y> et <~y => ~x> sont interchangeables.
    • De Morgan : <~x & ~y> et ~<x V y> sont interchangeables.
    • Echange : <x V y> et <~x => y> sont interchangeables.

    TNT

    • Termes
      • Symboles numériques : 0, 1 = S0, 2 = SS0, 3 = SSS0 ...
      • Variables : a, b, c, d, e, a', ...
      • Termes s, t -> Ss, (s+t), (s.t)

    • Propositions
      • Atomes propositionnels : s = t
      • propositions x, y -> ~x, <x & y>, <x V y>, <x => y>, Eu:x, Au:x

    • Règles
      • Règles du calcul propositionnel
      • Axiomes
        • Aa:~Sa=0
        • Aa:(a+0)=a
        • Aa:Ab:(a+Sb)=S(a+b)
        • Aa:(a.0)=0
        • Aa:Ab:(a.Sb)=((a.b)+a)
      • Nouvelles règles de la TNT : quantificateurs
        • Spécification :
          • Au:x |- x
          • Au:x |- [u->y]x substitution de u par y dans x, si y ne contient aucune variable quantifiée dans x.
        • Généralisation : x |- Au:x si u est libre dans x. Restriction : dans une fantaisie, aucune généralisation d'une variable libre dans la prémisse de cette fantaisie n'est permise.
        • Interchangeabilité : Au:~et ~Eu: sont interchangeables dans un théorème.
        • Existence : Si un terme (qui ne contient pas de variable non libre) apparait dans un théorème, il est possible de le remplacer par une nouvelle variable en plaçant E en tête du théorème.
      • Règles d'égalité et de succession
        • Symétrie : r=s |- s=r
        • Transitivité : r=s, s=t |- r=t
        • Ajout de S : r=t |- Sr=St
        • Suppression de S : Sr=St |- r=t
